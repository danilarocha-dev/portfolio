2024

Réplica da Interface do Spotify

Projeto desenvolvido para praticar HTML5, CSS3 e organização de layout complexo inspirado na interface do Spotify.

Saiba mais...

Este projeto evidencia a construção de uma interface moderna e bem estruturada, utilizando HTML5 semântico e CSS3 modularizado, com separação de arquivos para facilitar manutenção e escalabilidade. O layout foi organizado em componentes claros (sidebar, biblioteca, conteúdo principal e playlists), garantindo melhor legibilidade do código. Foram aplicados media queries, reset CSS, variáveis CSS e arquivos específicos para cada área da aplicação. Além disso, o projeto já conta com a estrutura inicial de APIs simuladas (JSON), preparando o ambiente para integração dinâmica de dados e evolução futura com JavaScript.
